# Selamat Sempro Sayang ❤️
A responsive single-page website created as a digital congratulatory gift for a university seminar proposal milestone.

## Features
* Multi-section interactive experience
* Personal congratulatory messages
* Memory gallery with photos
* Responsive design for desktop and mobile devices
* Background music support
* Interactive gift surprise animation
* Smooth navigation between sections

## Technologies Used
* HTML5
* CSS3
* JavaScript

## Purpose
This project was developed as a creative web-based greeting card to celebrate an important academic achievement and demonstrate the use of front-end web technologies for personalized digital experiences.
